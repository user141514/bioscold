## Gap Summary

**Gap 1 — Leakage-controlled fragment replacement benchmarking.** Every existing MMP-based ranking evaluation uses random or temporal splits that allow training fragments to appear in test candidate sets, inflating scores through fragment co-occurrence shortcuts. The transform-heldout construction is the first to isolate the ranking task from this confound, establishing a more rigorous evaluation standard. The gap is that this benchmark currently covers only one MMP database (ChEMBL37K); extending to SwissBioisostere, NeBULA, or CReM-generated databases would test whether the leakage problem is universal and whether candidate-level ranking transfers across MMP sources.

**Gap 2 — Candidate-level feature engineering for ranking, not generation.** All deep learning approaches to bioisosteric replacement (DeepBioisostere, GraphBioisostere, Ertl DNN) focus on improving candidate *generation* or pairwise *similarity scoring*, then rank by model confidence. None frames the problem as a supervised ranking task with hand-engineered features describing the *candidate replacement itself* (structural, physicochemical, provenance). This gap is the most promising because the 77-feature HistGB scorer demonstrates that explicit feature engineering can outperform generative model confidence ranking, and the approach is model-agnostic (works with any candidate generator).
