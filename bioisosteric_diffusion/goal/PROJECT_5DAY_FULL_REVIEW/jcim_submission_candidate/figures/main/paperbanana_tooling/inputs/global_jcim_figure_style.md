Use a restrained JCIM / cheminformatics manuscript style.

Global visual rules:
- White background.
- One sans-serif font family: Arial, Helvetica, or DejaVu Sans.
- Navy (#0B3F99) for task/base evidence.
- Teal (#087A74) for heldout/main method/post-audit/triage route.
- Red (#B31F24) for leakage/removed/alert.
- Gray (#222222/#666666) for neutral text.
- Light gray (#D9D9D9) for panel borders and gridlines.
- Very light fills only: light blue (#F4F8FF), light teal (#F2FBFA), light red (#FFF6F4).
- No gradients, shadows, decorative icons, dashboard/KPI styling, or saturated colors.
- Thin arrows, thin panel borders, compact labels, and strong alignment.
- Caption should carry explanation; figure text should be short.

Important workflow rule:
PaperBanana outputs should be treated as concept/style drafts. The final manuscript figures must remain editable SVG/PDF/PNG assets with locked scientific content.
