Global style for all figures:

- Target: JCIM / cheminformatics manuscript figure, with modern ML schematic discipline.
- White background.
- Sans-serif font: Arial / Helvetica / DejaVu Sans.
- Navy #0B3F99 = task/base evidence.
- Teal #087A74 = heldout/main method/post-audit/triage route.
- Red #B31F24 = leakage/removed/alert.

- Light gray #D9D9D9 = panel borders and gridlines.

- No gradients, no shadows, no decorative icons, no dashboard/KPI style.
- Keep labels compact; do not put paragraph explanations inside the figure.
- Prefer clean grids, aligned modules, thin arrows, and strong caption support.
