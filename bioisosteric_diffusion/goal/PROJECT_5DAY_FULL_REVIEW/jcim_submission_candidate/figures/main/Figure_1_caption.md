Figure 1. Leakage-controlled closed-vocabulary fragment replacement ranking. (A) Each query is defined by an old fragment and attachment signature, and the task is to rank an attachment-compatible closed vocabulary of replacement candidates so that observed replacements are recovered in the Top-10. (B) A random split can place the same transform identity, represented as (f_old, sigma), in both training and evaluation data, creating a leakage route. The transform-heldout split prevents this by enforcing zero transform-identity overlap across splits. (C) Evaluation tiers are separated by role: training builds the vocabulary and features, development/calibration tunes the protocol, the secondary blind set supports the primary claims, and canonical analysis is reserved for robustness checks.
