Supplementary Figure S4. Categorical schema alignment audit. Naive one-hot encoding fitted independently on the development/calibration and secondary blind candidate matrices can assign different categorical meanings to the same column position, producing misaligned blind predictions. The frozen schema repair fits the categorical template on the development/calibration set, maps unseen blind categories to OTHER, and preserves column order before prediction. The Top-10 change from 0.7120 to 0.8851 is an implementation repair effect, not a model contribution.
